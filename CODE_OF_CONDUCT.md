# Code of Conduct

This project expects respectful, technical collaboration.

## Expected Behavior

- Discuss behavior, interfaces, and evidence directly.
- Critique code and design decisions, not people.
- Document environmental blockers and reproduction steps precisely.
- Respect the repository rule that Rust code here is written from scratch.

## Unacceptable Behavior

- Personal attacks, harassment, or intimidation
- Deliberate misrepresentation of test results or benchmarks
- Submitting copied Rust source from other projects without explicit approval

## Reporting

Open a private report with the repository maintainers if a discussion stops
being productive or safe.
